Role: Static Web Developer / Portfolio Website Creator

Tech Stack: HTML, CSS, JavaScript (no frameworks or libraries)

Objective:
Design and develop a clean, responsive, and visually appealing personal portfolio website using only HTML, CSS, and JavaScript. The site should reflect a professional image suitable for showcasing my work as a static web developer and help me attract internship and job opportunities.

Required Pages:

Home Page

Welcome message with a brief introduction

Call-to-action buttons (e.g., “View My Work”, “Contact Me”)

Smooth scrolling navigation

Resume Page

Display my resume in a clean, readable layout

Option to download the resume as a PDF

Skills Page

List technical skills (e.g., HTML, CSS, JavaScript, etc.)

Use visual elements like progress bars, icons, or cards to represent proficiency levels

Projects Page

Showcase individual projects with:

Project title

Short description

Technologies used

Live demo link (if available)

GitHub repository link (mandatory for each project)

Use cards or a grid layout for clean presentation

Experience Page

Highlight internship experience

Include company name, role, duration, and key responsibilities/achievements

Contact Page

Contact form (name, email, message) with basic client-side validation using JavaScript

Display my email address and links to professional profiles (e.g., LinkedIn, GitHub)

Design & Theme Requirements:

Theme: Light theme with mild, soft, and professional colors (e.g., pastel blues, light grays, off-whites)

Layout: Clean, modern, and minimalist

Responsiveness: Fully responsive across mobile, tablet, and desktop devices

Navigation: Sticky or fixed navigation bar with smooth scrolling to sections

Interactivity: Subtle hover effects, smooth transitions, and JavaScript-enhanced user experience (e.g., form validation, dynamic content)

Deliverables:

Complete source code (HTML, CSS, JS files)

Well-organized file structure

Comments in code for clarity

Ready-to-deploy static site

Goal:
Create a top-tier portfolio website that effectively showcases my skills, projects, and experience while providing an excellent user experience and reflecting attention to detail and professionalism.
